üì¶ How to Use Google Fonts in Flutter

1. Add to pubspec.yaml:

dependencies:
  google_fonts: ^6.1.0

2. Import in your Dart file:

import 'package:google_fonts/google_fonts.dart';

3. Use in Text widgets:

Text(
  'Your Text Here',
  style: GoogleFonts.poppins(fontWeight: FontWeight.bold),
)

üé¨ How to Use Lottie in Flutter

1. Add to pubspec.yaml:

dependencies:
  lottie: ^3.0.0

2. Import in your Dart file:

import 'package:lottie/lottie.dart';

3. Use Lottie widget:

Lottie.network(
  'https://example.com/animation.json',
  width: 200,
  height: 200,
  fit: BoxFit.contain,
)

Or use local asset:

Lottie.asset('assets/animations/loading.json')

üß≠ How to Pass Arguments with Navigator

1. Create a model class (optional for complex data):

class Product {
  final String name;
  final double price;
  Product({required this.name, required this.price});
}

2. Navigate with arguments:

Navigator.push(
  context,
  MaterialPageRoute(
    builder: (context) => DetailPage(
      product: Product(name: 'Item', price: 99.99),
    ),
  ),
);

3. Receive arguments in destination page:

class DetailPage extends StatelessWidget {
  final Product product;
  DetailPage({required this.product});
  // Use product.name and product.price
}

üñºÔ∏è Network vs Asset Images

Network Image:

Image.network(
  'https://example.com/image.jpg',
  width: 200,
  height: 200,
  fit: BoxFit.cover,
  loadingBuilder: (context, child, progress) {
    return progress == null ? child : CircularProgressIndicator();
  },
)

Asset Image (add to pubspec.yaml first):

# pubspec.yaml
flutter:
  assets:
    - assets/images/
 Image.asset(
  'assets/images/photo.png',
  width: 200,
  height: 200,
  fit: BoxFit.cover,
)
